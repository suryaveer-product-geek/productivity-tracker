**🧠 Productivity Tracker**
**🚀 Overview**

This is a lightweight productivity tracking system built to solve a personal but common problem: losing structure while working from home.

Instead of relying on memory or manual discipline, this tool is designed to nudge consistent behavior through a habit-forming loop.

**🎯 Problem**

While working from home, I struggled with:

Losing track of time across the day
Working in bursts without clear structure
Failing to maintain manual productivity logs

I initially tried a manual digital tracker, but it failed because:

It required conscious effort every hour
No trigger → no habit formation
High friction → low consistency
**💡 Solution**

I built a Python-based productivity tracker that integrates into my daily workflow.

The goal was not just tracking — but behavior design:

Make logging productivity a default action, not a remembered task.

**🔁 Habit-Forming Loop Design**

This system is designed using a simple behavioral loop:

Trigger: Start of system / beginning of work session
Action: Log current task/activity
Reward: Sense of control + visible progress

By embedding the tracker into the system start, the dependency on memory is reduced.

**⚙️ How It Works**
Runs via Python scripts
Prompts user to log activity at defined intervals
Stores entries for later review
Creates a lightweight log of daily effort distribution

**📊 Metrics That Matter**

If scaled as a product, I would track:

-Daily Active Usage (DAU)
-Logging consistency rate (% of expected logs completed)
-Drop-off points (time of day when logging stops)
-Average entries per user per day



**🧪 Key Insight**

The core problem was not tracking, but remembering to track.

This project focuses on:

Reducing cognitive load instead of increasing discipline.
